Install the Spectral Phasor Package to ImageJ:

1. Copy the 'Spectral+phasor' folder provided in this directory

2. Navigate to the home directory of ImageJ and open the 'plugins' folder

3. Paste 'Spectral+phasor'

4. Restart ImageJ

Note: Fiji has compatibility issues with the plugin, make sure to use ImageJ


-------------------------------------------------------------------
Use SRS_Phasor_Dataset for demonstration.

Raw image: SRS_SKOV3_CH_2_corrected.tif

Phasor_RoiSet is the zip file containing example ROIs for the phasor plot. Directly drag and drop the file to ImageJ to open.
